ElevenLabs UI Vue is a component library and custom registry built on top of [shadcn-vue](https://shadcn-vue.com) to help you build multimodal agentic experiences faster.
It provides pre-built components for agents, transcription, audio & more.

Components are available via the `elevenlabs-ui-vue` command.

```bash
npx elevenlabs-ui-vue@latest add <component>
```

For example, to install the [Orb](/docs/components/orb) component, you can run:

```bash
npx elevenlabs-ui-vue@latest add orb
```

You can explore more blocks [here](/blocks) or view the [source code](https://github.com/vuepont/elevenlabs-ui-vue) on GitHub.
